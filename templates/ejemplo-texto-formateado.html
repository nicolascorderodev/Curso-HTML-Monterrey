Programación Web

La programación Web que se ha popularizado mucho en los últimos años. El crecimiento del internet ha permitido la creación de millones de páginas web, que ofrecen funcionalidades y servicios.

Lenguajes mas usados para la programación web

Los lenguajes mas usados para la programación web son HTML, CSS, JavaScript.

  • HTML

    • Este lenguaje le da estructura a las páginas.

  • CSS

    • Este lenguaje es para decorar páginas.

  • JavaScript

    • Este lenguaje es para manipular elementos de las páginas.
