Enlaces o hipervinculos son elementos muy comunes que permiten navegar hacia paginas ajenas a las nuestras o tambien sirven para navegar en nuestros archivos. TEMA 13: OTROS ELEMENTOS

Practicamente ya sabemos todos los elementos necesarios para construir una pagina web, en esta sección vamos a mirar otros elementos html útiles: iframes, tablas y spans. También aprenderás sobre las entidades, que sirven para agregar caracteres especiales a tu código.

Iframes

Los iframes se pueden visualizar como un marco que contiene otra página web tanto interna como externa a nuestro sitio, se debe hacer uso de la etiqueta iframe y el atributo src y se pueden agregar los atributos width y height para modificar ancho y alto.
