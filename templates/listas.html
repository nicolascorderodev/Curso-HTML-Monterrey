Listas en HTML(HyperText Markup Language)

Las listas se pueden usar para enumerar elementos o enlistarlos con viñeta. Hay 3 tipos y los veremos al avanzar con el documento.

Listas ordenadas

Las listas ordenadas se caracterizan por su orden, y su etiqueta es ol por cada elemento de lista se debe añadir la etiqueta li. El atributo type especifica el tipo de numeración que se quiere usar.

Ejemplo

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  4. Cuarto elemento
    1. Otro primer elemento
    2. Otro segundo elemento.

Listas Desordenadas

Son listas que no tienen orden, al igual que las listas ordenadas cada elemento debe estar dentro de la etiqueta li y este tipo de lista no tiene atributo type por lo cual se puede utilizar el atributo style pero ese atributo es mejor utilizarlo en un archivo aparte de CSS y mantener el orden de los archivos.

Ejemplo

  • Café
  • Leche
  • Pan
  • Huevos
  • Verduras
    • Lechuga
    • Tomate
    • Cebolla
