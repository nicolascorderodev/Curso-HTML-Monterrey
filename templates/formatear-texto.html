¿Que significa formatear texto en HTML(HyperText Markup Language)

Formatear texto significa darle otro estilo al texto para enfatizarlo, ya sea una palabra o una oración completa, esto no es recomendable ya que recordemos que HTML se usa para estructurar no para estilar, lo recomendable para modificar el estilo de un texto sea negrita, cursiva y subrayado es usando CSS(Cascading Styles Sheets)
