Bienvenidos a la introducción de HTML(HyperText Markup Language)

HTML forma el esqueleto de todas las páginas web. Este lenguaje tiene todas las herramientas para definir el formato de las paginas y permite incluir imágenes, formularios, botones e incluso enlaces a otros documentos.

Nuestros navegadores interpretan los documentos HTML para que podamos visualizarlos dandole clic derecho y dar clic en Inspeccionar o ver código fuente de la pagina.

Las competencias que adquirás en este módulo son las siguientes:

  • Reglas generales de HTML.
  • Conocimiento sobre elementos de HTML como encabezados, listas, parrafos, formularios, etc.
  • Estructuramiento basico de paginas web.
